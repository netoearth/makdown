### 运行

我们以管理员的方式打开cmd命令行，然后进入Nexus的可运行文件所在目录，我的是在D:\\Program\\nexus-3.18.1-01-win64\\nexus-3.18.1-01\\bin路径下

```
C:\WINDOWS\system32>d:D:\>cd D:\Program\nexus-3.18.1-01-win64\nexus-3.18.1-01\bin
```

然后我们可以先执行nexus -help命令查看下Nexus命令行常用指令

可以看到使用方法很简单，就如下几个指令

-   /install：安装
-   /uninstall：卸载
-   /stop：后台运行（关闭cmd后仍在后台运行）
-   /start：停止后台运行
-   /status：查看运行状态
-   /run：前台运行（可在cmd命令行实时查看log，需用CTRL+C停止运行，直接关闭命令行窗口也可停止运行）
-   /run-redirect：这个也可以前台运行，具体和/run的区别我也不是很清楚，谁知道的评论里说下

```
D:\Program\nexus-3.18.1-01-win64\nexus-3.18.1-01\bin>nexus -helpUsage: nexus.exe /install | /uninstall | /stop | /start | /status | /run | /run-redirect
```

安装

```
D:\Program\nexus-3.18.1-01-win64\nexus-3.18.1-01\bin>nexus /installInstalled service 'nexus'.
```

后台运行

```
D:\Program\nexus-3.18.1-01-win64\nexus-3.18.1-01\bin>nexus /startStarting service 'nexus'.
```

### 预览

通过nexus /start或则nexus /run运行程序后，在浏览器输入[http://localhost:8081](http://localhost:8081/)就可以预览软件的界面及功能了

![](https://img-blog.csdnimg.cn/20190903102016927.png?x-oss-process=image/watermark,type_ZmFuZ3poZW5naGVpdGk,shadow_10,text_aHR0cHM6Ly9ibG9nLmNzZG4ubmV0L21xZHhpYW94aWFv,size_16,color_FFFFFF,t_70)

默认访问端口是8081，当然我们也可以自定义端口，打开D:\\Program\\nexus-3.18.1-01-win64\\nexus-3.18.1-01\\etc\\nexus-default.properties文件，内容如下，修改application-port配置的值即可修改访问的端口

```
application-port=8081application-host=0.0.0.0nexus-args=${jetty.etc}/jetty.xml,${jetty.etc}/jetty-http.xml,${jetty.etc}/jetty-requestlog.xmlnexus-context-path=/nexus-edition=nexus-pro-editionnexus-features=\ nexus-pro-featurenexus.hazelcast.discovery.isEnabled=true
```
